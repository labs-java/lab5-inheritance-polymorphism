## Getting Started

### Prerequisites

- Maven 3.9.8
- Java 22

### Installing

Clone git repo:
```bash
git clone https://github.com/labs-java/lab5-inheritance-polymorphism.git
```

### Executing program

Ensure that you are in the project directory then run command below

```bash
mvn clean install
```

Move in target directory

```bash
cd .\target
```

Run the program

```bash
java -jar .\Lab_5-1.0-SNAPSHOT.jar
```
